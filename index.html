Neural Networks

B.E. Computer Science Thesis · Coimbatore Institute of Technology · 2025–2026

Interactive PNN Architecture — hover to explore data flow

Abstract

This work proposes a class of neural networks in which every computational unit is expressed as an explicit polynomial function, yielding models whose predictions are fully decomposable into interpretable terms. Unlike conventional "black box" networks, White Box PNNs expose the exact mathematical transformation at each layer, enabling domain experts and regulators to audit, verify, and trust model outputs without reliance on post-hoc explanation methods.

01

Structural Transparency

Each neuron's computation is an explicit polynomial — the model's decision path is fully traceable from input to output, eliminating the need for approximation-based interpretability tools like LIME or SHAP.

02

Learnable Polynomial Activations

Traditional activation functions are replaced with degree-n polynomial expansions whose coefficients are learned during training, providing richer representational capacity while maintaining closed-form expressions.

03

Decomposable Predictions

Every output can be algebraically decomposed into contributing polynomial terms — allowing researchers and regulators to quantify the influence of each input feature on the final prediction.
